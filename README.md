# Snake Game

Ce projet est un jeu Snake dans le terminal. Le joueur contrôle le snake au clavier, mange la nourriture pour augmenter sa taille et son score, et la partie se termine en cas de collision.

## Fonctionnalités
- Menu principal :  **Nouvelle partie (N)** / **Quitter (Q)**
- Déplacement avec les flèches du clavier
- Nourriture `*` générée aléatoirement
- Snake qui grandit quand il mange
- Fin de partie **Game Over** en cas de collision avec les murs ou avec le corps du snake
- Score affiché + record stocké dans `score.txt`


## Structure

├─ Snake_Game.py

└─ score.txt  # généré après une partie (record)
